# numericaljs
Numerical methods and algorithms for Javascript.

This library exposes Numerical Methods functions for JavaScript. 

## Linear Algebraic Equations. 

### Direct Methods. 

- Gauss Elimination Method.
- Dolittle's decomposition method. 
- Cholesky's decomposition method.

### Iterative Methods.

- Conjugate Gradient.
- Gauss Seidel.
